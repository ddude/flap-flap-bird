# flap-flap-bird

A very simple clone of Flappy Bird I made at work to showcase Clojure.

The assets were taken from [Floppy Bird](https://github.com/nebez/floppybird/).

## Usage

Launch and have fun!

Use mouse clicks to jump and space to rewind time.
